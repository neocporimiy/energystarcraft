## 📖 About

**Energy Starcraft** adds the **Energy Forge** — a machine that converts raw energy into Nether Stars. Feed it **350,000,000 FE** from any energy mod (Mekanism, AE2, Thermal Series, Immersive Engineering, etc.) and watch Nether Stars appear automatically. No boss fights, no wither farms — just pure industrial power.

---

## ✨ Features

- 🔨 **Energy Forge** — a new block that automatically crafts Nether Stars from FE
- ⚡ **350,000,000 FE = 1 Nether Star** — pure energy-to-item conversion
- 🔌 **Universal FE compatibility** — works with Mekanism, AE2, Thermal, IE and more
- 🎨 **Custom animated GUI** — with charge bar, floating particles and smart tooltips
- 🌍 **Fully translated** into English and Russian
- 📦 **Lightweight** — single block, no bloat

## 🛠️ Installation

1. **Download** the latest release from [Releases](../../releases) or [Modrinth](https://modrinth.com/mod/energy-starcraft)
2. **Install** [NeoForge 21.11.42+](https://neoforged.net/)
3. **Drop** `energystarcraft-*.jar` into your `mods` folder
4. **Enjoy!** 🎉

---

## 🎮 How to Use

1. Craft the **Energy Forge** and place it in the world.
2. Connect any FE cable/conduit to any side of the block.
3. Wait until it accumulates **350,000,000 FE** — a Nether Star will appear in the output slot.
4. Take the stars and keep the energy flowing! ⚡

## 🔧 Building from Source

```bash
# Clone the repository
git clone https://github.com/neocporimiy/energystarcraft.git
cd energystarcraft

# Build the mod
./gradlew build
```

> 📦 The compiled jar will be located at `build/libs/energystarcraft-*.jar`
